# Utilyx.app - Product Roadmap Q2-Q3 2026

## Vision
Plateforme d'outils en ligne gratuits (freemium) ciblant le trafic organique via SEO.
Positionnement unique: **UX premium + pas de watermark + pas de limites** (gap entre SmallPDF et PDF24).

## SEO Requirements par outil
Chaque outil doit inclure:
- **Title tag**: `[Keyword] | [Benefit] | Utilyx` (pattern SmallPDF-inspired)
- **Meta description**: 150-160 chars, action-oriented, "free" keyword
- **FAQ Schema**: 5-8 questions-réponses (People Also Ask targeting)
- **How-To Schema**: étapes numérotées
- **Hreflang**: fr, en, es, de, ar, pt

## KPIs
- Trafic organique: objectif 500K sessions/mois fin Q3
- Taux de conversion freemium: 2-3%
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- Bounce rate: < 40%
