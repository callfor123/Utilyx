# Utilyx - Analyse Concurrentielle PDF Tools

## 1. SmallPDF (smallpdf.com)
- **Positionnement**: "Make PDF Easy" - UX premium, freemium agressif
- **Outils phares**: Compress PDF, PDF to Word, Merge PDF, eSign PDF
- **Modèle**: 2 tâches gratuites/jour, puis $12/mois
- **SEO**: Domine "compress pdf", "pdf to word" en EN
- **Points faibles**: Paywall très restrictif, watermark sur exports gratuits
- **DA**: ~80 | **Trafic**: ~50M/mois

## 2. iLovePDF (ilovepdf.com)
- **Positionnement**: "Every tool you need to use PDFs" - suite complète
- **Outils phares**: Merge, Split, Compress, Convert, Sign
- **Modèle**: Gratuit avec limites, Premium $6.60/mois
- **SEO**: Fort sur "merge pdf", "split pdf" multilingue
- **Points faibles**: UI datée, publicités dans version gratuite
- **DA**: ~85 | **Trafic**: ~80M/mois

## 3. PDF24 (tools.pdf24.org)
- **Positionnement**: 100% gratuit, pas de limites
- **Outils phares**: Create PDF, Compress, Merge, Convert
- **Modèle**: Entièrement gratuit (monétisation via ads)
- **SEO**: Forte présence longue traine, beaucoup de pages
- **Points faibles**: Design basique, UX confuse, trop d'options
- **DA**: ~75 | **Trafic**: ~25M/mois

## 4. TinyPNG (tinypng.com)
- **Positionnement**: "Smart PNG and JPEG compression"
- **Outils phares**: Image compress (PNG, JPG, WebP)
- **Modèle**: 20 images gratuites, then $25/an
- **SEO**: Domine "compress png", "compress image"
- **Points faibles**: Un seul outil, pas de suite
- **DA**: ~80 | **Trafic**: ~15M/mois

## Opportunités Utilyx

### Gap #1: Pas de suite complète 100% gratuite haute qualité
SmallPDF et iLovePDF ont des paywalls restrictifs. PDF24 est gratuit mais UX médiocre.
**→ Utilyx: gratuit + UX premium = positionnement unique**

### Gap #2: i18n sous-exploitée
Seul iLovePDF propose vraiment du multilingue. SmallPDF est EN-centré.
**→ Utilyx: 6 langues dès le lancement = avantage SEO massif**

### Gap #3: Outils image + PDF dans une seule plateforme
Chaque concurrent est soit PDF soit Image, pas les deux.
**→ Utilyx: plateforme unifiée = cross-selling naturel**

### Gap #4: Calculators et générateurs orphelins
Ces outils sont dispersés sur des sites pauvres.
**→ Utilyx: hub complet = autorité de domaine cumulative**
